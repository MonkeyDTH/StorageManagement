# 家庭库存管理系统

## 项目简介
这是一个基于Flask开发的家庭库存管理系统，用于帮助用户管理和追踪家庭物品的购买、存储和销售情况。系统提供了直观的Web界面，支持物品的添加、编辑、分类管理以及数据统计功能。

## 主要功能
- **物品管理**
  - 添加新物品（包含名称、类别、购买价格、数量等信息）
  - 编辑现有物品信息
  - 支持物品图片上传和自动压缩
  - 记录物品的购买渠道、状态和备注信息

- **分类系统**
  - 支持主分类和子分类的两级分类结构
  - 灵活的分类筛选功能
  - 分类统计和数据汇总

- **财务追踪**
  - 记录购买价格和运费
  - 追踪销售价格和日期
  - 自动计算总投资和收益

- **数据统计**
  - 物品库存总览
  - 购买和销售金额统计
  - 分类数据分析

## 技术栈
- **后端框架**：Flask
- **前端技术**：HTML、CSS、JavaScript
- **数据存储**：JSON文件
- **图片处理**：Pillow

## 安装使用
1. 克隆项目到本地：
```bash
git clone [项目地址]
cd StorageManagement
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 运行项目：
```bash
python run.py
```

4. 在浏览器中访问：
```
http://localhost:5000
```

## 数据存储
系统使用JSON文件存储物品数据，数据文件位于 `app/data/items.json`。每个物品记录包含以下信息：
- 物品ID
- 名称
- 类别
- 购买价格
- 购买日期
- 数量
- 图片
- 购买渠道
- 物品状态
- 运费
- 销售信息（如果已售出）
- 备注

## 注意事项
- 首次运行时，系统会自动创建必要的目录和数据文件
- 上传的图片会自动压缩以节省存储空间
- 建议定期备份数据文件
